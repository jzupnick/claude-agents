# Tools

Executable utilities. Scripts, commands, helpers.

## What Goes Here

- Scripts that wrap complex operations
- CLI tools with specific purposes
- Integration helpers
- Code generators
- Testing utilities

## Principles

- One tool, one job
- Compose over configure
- Fail fast and loud
- Document the happy path only

## Template

```bash
#!/usr/bin/env bash
# tool_name.sh
# 
# What: One line description
# Why: The problem this solves
# Usage: ./tool_name.sh [args]

set -euo pipefail

# Your code here
```
