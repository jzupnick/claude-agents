# Examples

Real usage. Show, don't tell.

## Structure

Each example includes:
- The problem you were solving
- What you tried first (and why it didn't work)
- The final solution
- Key learnings

## Format

```
examples/
  project-name/
    problem.md          # What you were trying to do
    solution.md         # What worked
    agents-used.md      # Which agents/tools you combined
    learnings.md        # What you'd do differently
```
