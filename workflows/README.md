# Workflows

Multi-step processes that solve recurring problems.

## What Goes Here

Processes you found yourself repeating 3+ times:
- Feature development cycles
- Code review processes
- Deployment sequences
- Debugging protocols
- Refactoring patterns

## Format

```markdown
# Workflow Name

## Trigger
What starts this workflow?

## Steps
1. First thing (tool/agent/manual)
2. Second thing (tool/agent/manual)
3. Third thing (tool/agent/manual)

## Decision Points
Where do you branch? What are the criteria?

## Output
What's the deliverable?

## Time Investment
How long does this actually take?
```
