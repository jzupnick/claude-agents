# Claude Code Agents
*Enhanced with Northwestern MPD2 Design Decision Methodologies*

A comprehensive design decision system that integrates Northwestern MPD2 academic frameworks with practical agents, tools, skills and workflows for Claude Code.

## Philosophy

- **Ship within a year**: No agent takes more than a year to ROI
- **Follow your doubt**: If it feels wrong, it probably is
- **Prototype the hard thing**: Test the differentiation early
- **Think slow, act fast**: Deep understanding, rapid execution
- **Academic rigor, practical application**: Northwestern MPD2 frameworks enhance decisions without slowing execution
- **Enhancement over replacement**: Build upon existing capabilities rather than starting from scratch
