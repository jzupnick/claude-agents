# product — Agent Skills

| Skill | Description |
|-------|-------------|
| `/product:brain-dump` | Interactive assistant for organizing and processing Obsidian brain dumps in the root folder |
| `/product:brainstorm` | Guide for collaborative design work before implementation. Use when creating or developing ideas, before writing code or implementation plans. Refines rough ideas into detailed specifications through structured dialogue. |
| `/product:brainstorm-partner` | Brainstorm Partner — ideation, divergent thinking, concepts |
| `/product:compliance` | Navigate regulatory requirements, certifications, and safety standards for product development. Use when planning certification strategy, preparing regulatory submissions, reviewing marketing claims, or assessing compliance risks. |
| `/product:concept` | Concept Developer — concept development, validation, MVPs |
| `/product:concept-evaluation` | Use when evaluating product concepts, writing design briefs, applying idea-to-prototype frameworks, or validating concepts against market needs |
| `/product:design-strategy` | Use when developing product design strategy, applying lean design principles, or evaluating design concepts against business goals |
| `/product:discovery` | Use when conducting market discovery, identifying opportunities, validating value propositions, or creating customer segment maps |
| `/product:ideation` | Idea Generator — idea generation, creative techniques |
| `/product:linear` | Master Linear project management with MCP integration. Use when auditing backlogs, syncing issue status with codebase reality, creating well-structured issues, or generating evidence-based sprint plans. |
| `/product:materials` | Select and evaluate materials based on performance, cost, sustainability, and manufacturing constraints. Use when choosing materials for product design, evaluating trade-offs, or assessing manufacturing compatibility. |
| `/product:portfolio-strategy` | Use when optimizing product portfolios, applying Northwestern MPD2 strategic frameworks, or calculating resource allocation for innovation pipelines |
| `/product:prioritization` | Use when prioritizing product features, grooming backlogs, applying RICE/ICE scoring, or making build-vs-buy decisions |
| `/product:prototype-plan` | Use when planning prototypes, designing validation experiments, creating MVPs, or running user testing |
| `/product:refine-notes` | Use when refining Northwestern MPD2 course notes into structured knowledge, searching academic content, or building learning references in Obsidian |
| `/product:release` | Coordinate the final push from validated product to production release. Use when planning launches, managing release checklists, coordinating go-to-market, or transitioning from development to operations. |
| `/product:stage-gates` | Use when managing structured product development through stage-gate processes, evaluating gate readiness, or making go/no-go decisions |
| `/product:vendor-management` | Manage vendor relationships, manufacturing partners, and supply chain operations. Use when sourcing vendors, qualifying suppliers, managing BOMs, coordinating tooling, or planning pilot production runs. |
